# Block-Blash
Merhabalar, bu projemde bir staj başvurusunda yapmam istenilen basit bir Çökme / Patlama mekaniğine sahip blok patlatma oyunu geliştirdim. Projem tamamen temel mantıkları içerir yani bir level tasarımı yada score edinme gibi mekanikler projede yer vermedim. Eğer herhangi bir sorunuz olursa profilimde yer alan mailden bana ulaşabilirsiniz.
